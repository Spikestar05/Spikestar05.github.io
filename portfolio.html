Projects

Build Your Ride (BYR)

I collaborated with a team to design and develop Build Your Ride (BYR), an interactive 3D car customization platform created for the Toyota track at HackUTD 25. The project was aimed to change vehicle personalization into a real-time, immersive experience by integrating advanced web rendering, data-driven recommendations, and AI-assisted interaction.

Github
  • Developed real-time 3D car customization platform using Three.js and Blender assets within Nuxt.js framework, enabling interactive vehicle personalization with optimized rendering performance for Toyota track competition
  • Engineered MongoDB-powered recommendation engine to match custom builds with Toyota models and implemented finance calculator providing real-time loan/lease payment estimates based on user configurations
  • Integrated ElevenLabs AI voice assistant for conversational guidance and designed scalable component architecture with seamless frontend-backend data flow using Git-based collaborative workflow

CometConnect - Research Opportunity Platform

I collaborated with a team to design and develop Comet Connect, a full-stack web application aimed at bridging the gap between students and faculty by centralizing research opportunity postings at UT Dallas.

Github
  • Engineered full-stack web application connecting UT Dallas students with faculty research opportunities, implementing secure RESTful APIs with MySQL for user authentication, profile management, and data persistence
  • Developed responsive frontend using React hooks for state management, featuring profile management with form validation and transaction management to ensure data integrity across database tables
